## 🛠️ Setup

```bash
# Clone the project to your machine
git clone https://github.com/Chai4523/movie-app.git
cd movie-app

# This project uses pnpm as package manager, install if you dont have it
npm install -g pnpm

# Install dependencies
pnpm install

# Run the development server
pnpm run dev
```
